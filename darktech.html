Scrolltrap

In dit project heb ik samengewerkt met meerdere medestudenten om een website te maken voor DarkTech, een informatieve scrolltrap website. Deze website is gemaakt voor Bits of Freedom, een organisatie die zich inzet voor digitale rechten en vrijheid op het internet.

Hier is de link naar ons Project.

Mijn rol in het project

Mijn rol in dit project lag meer bij het design en de front-end ontwikkeling. Ik heb als profiel front-end gekozen dus het merendeel van mijn werk bestond uit het maken van de HTML-structuur, user experience en het stylen met CSS. Daarnaast heb ik ook geholpen met het bedenken van het concept en de functionaliteiten van de website.

Skills in dit project

De skills die ik heb laten zien in dit project zijn onder andere:

  • HTML5 en CSS3 voor het bouwen en stylen van de website.
  • Samantisch werken zodat de goede HTML elementen worden gebruikt in de website.
  • Samenwerken in een teamomgeving, waarbij we elkaars werk hebben beoordeeld en feedback hebben gegeven.
  • Gebruik van versiebeheer met Git om onze code te beheren en samen te werken aan hetzelfde project.

Mijn leerdoelen

Tijdens dit project was mijn leerdoel de eerste 2 sprints om vaker te laten zien wat ik heb gemaakt en wat de groepsgenoten er van vonden. Halverwege het project kwam ik erachter dat dit al best goed ging waardoor ik mijn leerdoel heb aangepast. Mijn nieuwe leerdoel was om tijdens het project meer vragen te stellen aan mijzelf voor ik iets ging maken. Dit heb ik gedaan door voor ik begon met coderen eerst een klein plan te maken van wat ik wilde maken en hoe ik dat ging doen. Hierdoor kon ik beter inschatten of iets haalbaar was binnen de tijd en of het goed paste bij de rest van de website.
